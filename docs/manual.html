1. 소개서

작업자가 본인 휴대폰으로 휴가증을 직접 작성하면 클라우드에 자동 저장됩니다. 서무는 매일 한 번 사이트에서 일괄 취합 후 자동화 프로그램으로 그룹웨어에 등록합니다.

1.1 전체 흐름

  1. 작업자 휴대폰으로 사이트 접속 → 휴가증 작성
  2. 작성 즉시 클라우드 자동 저장
  3. 서무 매일 PC에서 사이트 접속 → [서버에서 불러오기] → [파일로 내보내기]
  4. 자동화 프로그램(run.bat)으로 그룹웨어 임시저장
  5. 그룹웨어 검토·신청 후 [처리 완료]

1.2 역할별 권한 (사번 로그인 기반)

※ 모바일은 어떤 사번이든 작업자 화면만 표시 (서무 작업은 PC 전용)

2. 매뉴얼 (작업자용)

2.1 접속 & 로그인

주소: https://djlee-cosmax.github.io/Vacation-Permit-Auto-System/

  • 사번 + 비밀번호 1234 입력 → [로그인]
  • 로그인 유지(30일) / 아이디 저장 / 👁 보기 토글 지원
비밀번호 변경 (필수): 첫 로그인 후 상단 [비밀번호 변경] → 숫자 6자리 이상 + 보안 질문/답변. 보안 질문은 비밀번호 찾기에 사용됩니다.

※ 비밀번호 분실 시 로그인 화면의 [비밀번호 찾기] 사용 (보안 질문 미등록은 관리자에게 초기화 요청)

2.2 홈 화면에 추가 (모바일 권장)

휴대폰 홈 화면에 앱 아이콘으로 추가하면 한 번 탭으로 실행됩니다.

  • iOS Safari — 하단 공유 버튼 → "홈 화면에 추가"
  • Android Chrome — 우상단 ⋮ → "홈 화면에 추가"
  • 홈 화면 앱에서 새로고침이 필요할 때는 상단 ↻ 아이콘 클릭

2.3 휴가증 작성

  1. 구분 선택 (연차 / 반차 / 반반차 / 생휴 / 하기휴가 / 결근 등)
  2. 개수 입력 (예: 연차 2개)
  3. 기간 시작일~종료일 선택
  4. 사유 입력
  5. [+ 추가] 클릭 → 클라우드 자동 저장

※ 한 사람이 여러 유형(예: 연차+반차)을 쓸 때는 휴가증을 따로 작성

※ 동일 휴가증이 이미 있으면 경고 — 정말 한 번 더 등록할 때만 [확인]

2.5 내 휴가증 조회

상단 [내 휴가증] → 이름 + 휴대폰 마지막 4자리 → 처리 완료된 본인 휴가증 표시 (최근 14일)

잘못 작성 시: 작성 직후 우측 카드의 [삭제]로 취소. 서무 처리 후엔 서무에게 별도 처리 요청.

3. 매뉴얼 (서무용)

3.1 로그인

주소: https://djlee-cosmax.github.io/Vacation-Permit-Auto-System/

  • 사번 122240096 + 비밀번호 1234 입력
  • 로그인 시 자동으로 서무 권한 부여 → 추가 버튼 표시
  • 모바일에서는 서무 권한 비활성 — PC에서만 사용

3.2 매일 사용 절차

  1. PC에서 사이트 접속 → 서무 사번으로 로그인
  2. 우측 카드의 [서버에서 불러오기] 클릭 (미처리 휴가증 + 최근 7일)
  3. 화면에서 휴가증 목록 검토
  4. 상단 [파일로 내보내기] → JSON + XLSX 다운로드
  5. 자동화 폴더의 run.bat 더블클릭 → JSON 선택 → 진행
  6. 완료 알림 확인 → 그룹웨어 검토 후 신청 처리
  7. 모두 완료 후 사이트에서 [처리 완료] 클릭
처리 완료의 안전망: 자동화 끝까지 완료된 후에만 [처리 완료]를 누르세요. 누르지 않은 휴가증은 다음 날 다시 불러올 수 있어 안전합니다.

3.3 자동화 최초 설치 (한 번만)

  1. Python 3.10~3.12 설치 — python.org/downloads ("Add Python to PATH" 체크)
  2. 자동화 폴더(또는 ZIP) 적당한 위치에 배치 (예: D:\휴가증자동화\) — ZIP은 풀기 전 우클릭 → 속성 → "차단 해제"
  3. setup.bat 더블클릭 → 자동 설치 완료 대기
  4. 첫 run.bat 실행 → 그룹웨어 로그인 → 이후 자동 유지
