# 휴가증 자동 반영 — 서무 담당자용 자동화

COSMAX COIN 그룹웨어에 휴가증을 자동으로 임시저장하는 도구입니다.

## 📦 구성 파일

| 파일 | 역할 |
|---|---|
| `setup.bat` | **최초 1회 실행** — Python 패키지 + Edge 드라이버 설치 |
| `run.bat` | **평소 사용** — 더블클릭으로 자동화 실행 |
| `automate.py` | 메인 자동화 스크립트 (Python + Playwright) |
| `requirements.txt` | 필요한 Python 패키지 목록 |
| `profile/` | (자동 생성) Edge 영구 프로필 — 쿠키 보관 |

## 🚀 최초 설치 (한 번만)

### 1. Python 설치
- https://www.python.org/downloads/ 에서 **Python 3.10 이상** 다운로드
- 설치 화면에서 **"Add Python to PATH"** 옵션을 반드시 체크
- 설치 완료 후 PC 재시작 (선택)

### 2. 본 폴더에서 `setup.bat` 더블클릭
- Playwright 패키지 + Edge 드라이버가 자동 설치됩니다
- 설치 진행 화면이 표시되며 완료까지 1~3분 정도 소요

### 3. 첫 실행 (로그인 등록)
- `run.bat` 더블클릭
- 파일 선택 다이얼로그가 뜨면 **취소**해도 됩니다
- (또는 휴가증 JSON 파일이 있으면 선택)
- Edge가 자동으로 열리고 그룹웨어 로그인 화면이 표시됨
- **사번 / 비밀번호로 직접 로그인** → 로그인 완료 후 자동화 진행
- 이후 실행부터는 로그인 자동 유지

## 📋 평소 사용 절차

1. 현장 작업자들에게서 휴가증 정보를 수집
2. 휴가증 사이트(https://djlee-cosmax.github.io/Vacation-Permit-Auto-System/)에 입력
3. 사이트 우측 상단 **"파일로 내보내기"** 버튼 클릭 → JSON 다운로드
4. 본 폴더의 `run.bat` 더블클릭
5. 파일 선택 다이얼로그에서 방금 다운로드한 JSON 선택
6. 확인 메시지가 뜨면 "예" 클릭
7. Edge가 자동으로 신청서를 차례로 임시저장
8. 완료 알림이 뜨면 그룹웨어에서 임시저장된 신청서들을 직접 검토 후 신청

## ⚠️ 주의사항

- 자동화는 **임시저장까지만** 진행합니다. 실제 신청은 서무 담당자가 검토 후 진행해야 합니다.
- 사내 네트워크 안에서만 동작합니다 (그룹웨어 자체가 사내 전용).
- 자동화 진행 중 Edge 창을 닫거나 다른 작업을 하면 오류가 발생할 수 있습니다.
- 자동화 완료 후 Edge 창은 직접 닫아 주세요 (결과 확인용으로 유지됨).

## ❓ 트러블슈팅

| 증상 | 해결 |
|---|---|
| `setup.bat`에서 Python 없음 에러 | Python 설치 후 PC 재시작. PATH 옵션 체크 확인 |
| `setup.bat`에서 패키지 설치 실패 | 사내 네트워크 프록시 문제일 수 있음. IT팀에 문의 |
| `run.bat`에서 Edge가 안 열림 | `setup.bat`을 다시 실행해서 Edge 드라이버 재설치 |
| 로그인 후에도 자동화가 진행 안 됨 | Edge 창을 그대로 두고 본 프로그램을 다시 실행 |
| 일부 신청서만 등록 실패 | 그룹웨어 페이지 변경 가능성. 담당자(djlee)에게 문의 |
| 자동화 실행 중 그룹웨어가 멈추거나 느림 | Edge 창의 PC 자원 상황 확인 후 재시도 |
